# あしあと仲人©️

あしあと仲人©️は、検索・即DM・人気順ではなく、  
表札街と立ち話から人となりをゆっくり読むための  
コミュニティSNS模型です。

これは HTML / CSS / JavaScript で作った静的UIデモです。  
ログイン、保存、通知、本物の足あと記録、本物の合図送信はありません。

## このデモで見てほしいこと

このデモの見せ場は、  
**人を条件で探すのではなく、部屋の立ち話を読み、ことばから人の気配に近づいていく流れ**です。

普通のSNSやマッチングアプリでは、先に人が並び、プロフィールや条件から相手を選びます。

あしあと仲人©️では、まず表札街を歩き、部屋の空気を見て、立ち話を読みます。  
その中で気になることばがあったら、肩書きカードへ進みます。

気になる人がいても、いきなりDMを送る必要はありません。  
そっと足あとを残す、合図をえらぶ、おばちゃんに言葉を整えてもらう。  
そのくらいの距離感から始めるための模型です。

---

## 表札ことばについて

このデモでは、最初に「今日、気になる表札ことば」を選びます。

これは、一般的なSNSのような条件検索ではありません。  
人を絞り込むためのタグではなく、表札街を歩くときに手に持つ小さなことばです。

気になることばを持って歩くことで、  
そのことばの気配が漂う部屋や、話の入口が少し見えやすくなります。

あしあと仲人©️では、  
人を条件で探すのではなく、  
ことばから部屋へ入り、立ち話から人の気配に近づいていきます。

## おすすめの歩き方

1. 今日、気になる表札ことばを選ぶ
2. 表札街へ歩く
3. 気になる部屋をのぞく
4. この部屋の立ち話を読む
5. 気になる発言から肩書きカードを見る
6. そっと足あとを残す
7. 合図をえらぶ
8. おばちゃんに言葉を整えてもらう

## このアプリにないもの

このデモには、以下の機能はありません。

- ログイン
- 保存
- 通知
- DM
- マッチング成立
- 本物の足あと記録
- 本物の合図送信
- 本人確認
- 部屋作成
- 自由なキーワード検索

これは本物サービスではなく、  
思想と導線を見るための静的UI模型です。

## 減圧導線について

あしあと仲人©️では、  
**減圧は、画面遷移の遅さではなく、言葉・導線・選択肢の少なさで表現しています。**

このデモには、リッチなアニメーションや派手な演出もほとんどありません。  
タップすると、画面はすぐに切り替わります。

その軽さの中で、  
立ち話から人の気配へ近づき、  
いきなり送らず、  
足あとや合図に留める流れを見てください。
