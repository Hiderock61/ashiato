# あしあと仲人©️ v0.3C-mini 明智版

v0.3Bの骨格を守りながら、言葉チューニングと迷子防止を入れた静的UI模型です。

## 反映したこと
- 「参加メンバー」ではなく「この部屋の住人」
- 「プロフィール」ではなく「肩書きカード」
- 「本人確認」ではなく「治安ゲート」
- 「共通肩書き」より「話の入口」を前面に表示
- スレッド本文に「コミュニティ名 ＞ スレッド名」を表示
- selectedMyTags が空のときの注意文
- 肩書きカードから 足あと / ノック / おばちゃん紹介文 へ進める導線
